whitematteranalysis

White matter tractography clustering, segmentation, registration

View on GitHub Download .zip Download .tar.gz

Synopsis

whitematteranalysis provides clustering and tractography analysis tools.

It implements algorithms from publications listed here: http://projects.iq.harvard.edu/whitematteranalysis/publications

Installation

1. Install python.

Anaconda is a nice option since it has VTK and scipy. First install anaconda, then run: 

  conda install vtk

2. Install the following python packages (dependencies).

Once you have anaconda installed, run: 

  pip install joblib

Note: If you decide to use another python that does not already have VTK, you can compile VTK.

  • VTK: http://www.vtk.org/Wiki/VTK/Building You will need to compile it with python wrapping.

3. Install whitematteranalysis into your python in the standard way.

 cd whitematteranalysis
 python setup.py install

4. Test the installation.

Run python, then:

>>> import whitematteranalysis as wma

This will produce errors if the required packages are not found.

5. Scripts for performing data analyses are located in whitematteranalysis/bin.

6. Please see the wiki for usage instructions.
